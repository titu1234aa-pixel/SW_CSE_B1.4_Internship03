HTML Image Tag - Variations and Practical Usage

This code explains practical and advanced variations of the HTML img tag. It includes clickable images, figure captions, decorative images, lazy loading, and image maps.

1. Image as a Hyperlink

An image can be placed inside an anchor tag to make it clickable. When the user clicks the image, the browser opens the linked page.

2. Image with Caption

The figure element groups the image and its related caption. The figcaption element provides a visible caption for the image.

3. Decorative Image

If an image is only for decoration and adds no content meaning, an empty alt attribute can be used.

4. Image with Lazy Loading

The loading attribute can delay image loading until the image is near the visible area of the page. This can improve page performance.

5. Image Map

An image map allows different areas of one image to act as different clickable links. The usemap attribute connects the image with the map element.

6. Meaningful Content Image

If the image carries information, the alt text should describe that information clearly.
