Position Property: dictates how an element is placed on a webpage. It controls whether an element stays in the normal page flow, or moves relative to its neighbours or locks it in a place when user scrolls 1. static: Normal page flow. (Default) 2. relative: Moves away from its normal spot without disturbing neighbours. 3. Absolute: Flies out of normalflow, positining itself inside its closest relative parent. 4. Fixed: Locks to the vieport. Stays in the exact same spot on the screen when scrolling. 5. Sticky: Scrolls naturally with the page, then locks in place like FIXED when hits the edge.

Both Display property and Position Property, controls layout, differnce is that display creates the box and position move the box.
